# eurosys-aegis.github.io

Website for **AEGIS (Grand Systems Challenges in the AI Era)** â€” a co-located workshop at **EuroSys 2026**.

This site is built using **Jekyll**, based on the MIT-licensed template: `yishanhe/jekyll-conference-template`.

## Local preview (optional)

1. Install Ruby + Bundler
2. Run:

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://127.0.0.1:4000/`.

## Update conference data

- `_data/conference.yml`: title, nav, organizers, etc.
- `_data/schedule.yml`: program timeslots
